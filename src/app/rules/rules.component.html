Game Rules

Welcome to Showdown – a competitive quiz game for two teams.

  1. The game is played in turns. At any given time, one team is considered to be in turn (active), while the other is passive.
  2. At the start of a turn, the passive team selects a category for the active team.
  3. Each category contains 6 questions. One of these questions includes a hidden bonus question, revealed only if the main question is answered correctly.
  4. The active team then picks a question from that category.
  5. After the question is opened, the active team can choose how to answer:
    • Open Answer – straight answer; worth 3 points if correct.
    • Ask for Tips – 4 possible answers are revealed and the active team chooses one of them; worth 1 point if correct.
  6. If the active team answers incorrectly, the passive team gets a chance to attempt a steal. A successful steal is worth 2 points.
  7. If the main question has a bonus and it was answered correctly, the bonus question is unlocked. Answering it correctly earns an additional 1 point.
  8. After the turn ends, teams switch roles and the game continues.
  9. The game ends when all questions have been answered. The team with the most points wins!

Good luck and have fun!
